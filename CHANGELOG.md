# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-04

### Added
- Initial release of ISight Sign Language Detector
- MediaPipe-based hand landmark detection with 21 3D points
- Multiple ML model support (Random Forest, SVM, Neural Network, Gradient Boosting)
- Real-time detection with confidence filtering and temporal smoothing
- Interactive data collection system with quality checks
- Comprehensive evaluation tools with accuracy metrics
- Visualization module with debugging features
- Quick start guide and comprehensive documentation
- Configuration module for centralized settings
- Custom exception classes for error handling
- Logging module for debugging and monitoring
- Utility functions module
- Constants module for project-wide values
- API documentation
- Troubleshooting guide
- FAQ documentation
- Performance benchmarks
- Security policy
- Contributing guidelines
- MIT License
- Automated setup script

### Features
- >95% target accuracy with comprehensive feature extraction
- Real-time performance at 55-60 FPS
- Support for multiple detector types
- Configurable confidence thresholds
- Cross-validation and hyperparameter optimization
- Confusion matrix and per-gesture accuracy analysis
- Temporal smoothing for stable predictions
- Extensive documentation and examples
