# Citation

If you use ISight in your research or project, please cite it as follows:

## BibTeX

```bibtex
@software{isight2026,
  author = {Blazehue},
  title = {ISight: High-Accuracy Sign Language Detection with MediaPipe},
  year = {2026},
  publisher = {GitHub},
  url = {https://github.com/Blazehue/Isight},
  version = {1.0.0},
  note = {A comprehensive, production-ready sign language detection system}
}
```

## APA Style

Blazehue. (2026). *ISight: High-Accuracy Sign Language Detection with MediaPipe* (Version 1.0.0) [Computer software]. GitHub. https://github.com/Blazehue/Isight

## MLA Style

Blazehue. *ISight: High-Accuracy Sign Language Detection with MediaPipe*. Version 1.0.0, GitHub, 2026, github.com/Blazehue/Isight.

## Chicago Style

Blazehue. 2026. "ISight: High-Accuracy Sign Language Detection with MediaPipe." Version 1.0.0. GitHub. https://github.com/Blazehue/Isight.
